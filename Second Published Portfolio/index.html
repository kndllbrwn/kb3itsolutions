View FAQs Page

Frequently Asked Questions Page

This page is a major concern, because it is often where students will come to get oriented to a site, especially if they are feeling overwhelmed. The existing page is so text heavy, it may not do a lot to alleviate that.
I did two things to make the page more inviting and user friendly. The first was to list all the questions at the top, so students can see all of the questions upon arrival, these questions are hyperlinked to anchers where the answers on the page are. I also made the right sided divs toggle allowing students to reduce the clutter if they want less text on the screen.

View Webcasts Page

Webcast Page

This is one of my more innovative designs, the existing page simply has the titles for videos and once you click them they open. I thought of a few ways to make this page more user friendly. The first update was to categorize the videos according to whatever professional development topic they were related to. I also added information about each of the videos to the title attribute of each link, allowing students to see a brief description of the video before opening it.
The most innovative addition to this page was a script I wrote that had an event listener for when the student clicked on the link. If it was triggered, the default behavior was modified, so instead of heading to the page the video was embedded in the current page.
